# Power BI Lab Activities

## Overview
This repository contains Power BI lab activities performed during workshop and college sessions. The activities cover data loading, transformation, DAX operations, data modeling, and dashboard visualizations using different datasets.

- Types of data
- Importing data from Web, CSV, TXT, and Excel
- Group By operations
- Conditional columns
- Custom columns
- Append & Merge tables
- Power Query transformations
- Restaurant dashboard
- Titanic dashboard

### Tools Used
- Power BI Desktop
- Power Query
- CSV / TXT / Excel / Web data
